🏥 Gallstone Risk & Metabolic Disorder Data Visualization Dashboard
📌 Overview

This project is an interactive data visualization dashboard designed to analyze and identify high-risk patients for gallstones and related metabolic disorders. The system transforms raw clinical data into meaningful visual insights to support better understanding of patient health patterns and risk factors.

The dashboard focuses on key medical indicators such as age, gender, BMI, cholesterol levels, and glucose trends to help highlight potential high-risk cases.

🎯 Key Features
📊 Age & Gender Analysis – Understand patient distribution across demographics
🪨 Gallstone Prevalence Visualization – Identify patterns in gallstone occurrence
⚖️ BMI Distribution – Analyze weight-related health risks
🧪 Cholesterol Insights – Track TC, LDL, HDL, and Triglycerides trends
🍬 Glucose Level Analysis – Monitor sugar-related risk indicators
🚨 High-Risk Patient Detection – Highlight patients with elevated risk factors
📈 Interactive Charts – Bar, Line, and Pie charts for dynamic exploration
🛠️ Tech Stack
⚛️ React.js – Frontend framework
📊 Recharts – Data visualization
🎨 CSS / UI Styling – For responsive and clean dashboard design
🧠 Data Processing – Clinical dataset analysis and transformation
